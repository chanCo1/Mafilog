# 매필로그(Mafilog)
여행 기록 관리 서비스

## 기술 스택
- **Framework**: Next.js 14 (App Router) / React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Radix UI
- **State Management**: Zustand, React Query
- **Form**: React Hook Form, Zod
- **Authentication**: NextAuth.js
- **Date**: date-fns, dayjs
- **HTTP Client**: Axios
- **Map**: react-kakao-maps-sdk

## 시작하기

### 설치
```bash
yarn install
```

### 개발 서버 실행
```bash
yarn dev
```

### 빌드
```bash
yarn build
```

### 프로덕션 실행
```bash
yarn start
```
